## Como subir

1. Extraia este ZIP.
2. Suba o CONTEÚDO da pasta para a raiz do repositório.
3. Não suba o ZIP fechado.
4. Em Settings > Pages, deixe:
   - Deploy from branch
   - Branch: main
   - Folder: /root
